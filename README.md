## 230515

### N-Queen 알고리즘

알고리즘 실행 시, 체스판의 상태가 변화(true->false, false->true)하는 좌표를 배열에 저장  
이 때 최적화를 위하여 좌표를 객체로 저장하지 않고 하나의 number 타입으로 암호화하여 저장  
예 : (12,13) : 1213, (4,5) : 405
